# Skills
 Job Portal für intelligente Recruiter

Wenn es Sie nervt, daß Sie nicht in der Lage sind, Ihren Bewerbern eine 
anständige Mail schreiben, in der Sie die Gründe für Ihre Absage erläutern, 
stattdessen schicken Sie inhaltsleere Textbausteine, dann kommt hier die Lösung.

Hier können Sie die Anforderungen für Ihre Jobs formulieren und die Bewerber 
Ihre Fähigkeiten. Die Software rechnet dann beides zusammen und kann den
Bewerbern, die nicht zum Zuge kommen, eine anständige Begründung schreiben und 
niemand ist beleidigt.

Beachten Sie: Die Software befindet sich noch in Entwicklung, nicht alle 
geplanten Features sind schon implementiert. Sie können aber schon mal einen 
Blick darauf werfen und, wenn Sie gute Ideen haben, können Sie gerne 
Verbesserungen vorschlagen.

Eine Installation finden Sie hier: https://skills.gaeckler.at/
